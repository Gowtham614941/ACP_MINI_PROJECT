Act as an expert C software engineer. Write a complete, modular, and well-documented C program for a 2D ASCII Graphics Editor. The editor must render shapes onto a 2D grid using '*' for lines/borders and '_' for the background. 

The implementation must adhere to the following strict requirements:

1. Data Structures & Canvas:
   - Define a fixed-size 2D character array canvas (e.g., 50 columns by 20 rows).
   - Implement an 'init_canvas()' function that fills the grid with the background character '_'.
   - Implement a 'display_canvas()' function that prints the current state of the grid to the console.
   - Define an 'Object' structure to represent shapes. Each object must track:
     * A unique ID (integer).
     * Type (ENUM: POINT, LINE, RECTANGLE, TRIANGLE, CIRCLE).
     * Geometry data (coordinates, radius, etc., using a union or specific struct fields).
   - Maintain an array of 'Object' structures to act as the scene graph (supporting up to 20 shapes).

2. Drawing & Rasterization Functions:
   Write functions to plot shapes onto the 2D array. Use mathematical algorithms appropriate for a discrete grid:
   - 'draw_line(int x1, int y1, int x2, int y2)': Implement using Bresenham's Line Algorithm.
   - 'draw_rectangle(int x, int y, int width, int height)': Plot the four bounding lines.
   - 'draw_circle(int cx, int cy, int radius)': Implement using Bresenham’s / Midpoint Circle Algorithm.
   - 'draw_triangle(int x1, int y1, int x2, int y2, int x3, int y3)': Plot three lines connecting the vertices.

3. Object Management (CRUD API):
   - 'add_object(...)': Instantiate a shape, assign it a unique ID, add it to the scene array, and re-render the canvas.
   - 'delete_object(int id)': Remove an object from the scene array by its ID and re-render the canvas from scratch.
   - 'modify_object(int id, ...)': Find an object by its ID, update its geometric properties (e.g., move it, change radius), and re-render the canvas.

4. Rendering Pipeline:
   - Implement a 'render_scene()' function. This function must clear the canvas to '_' and then iterate through the active objects array in order, calling the respective drawing functions for each object to bake them into the 2D array.

5. Code Architecture:
   - Ensure clean separation of concerns.
   - Provide a 'main()' function that demonstrates a complete workflow: initializing the canvas, adding a rectangle and a circle, displaying it, modifying the circle's size, deleting the rectangle, and displaying the final result.
   - Avoid external dependencies beyond <stdio.h>, <stdlib.h>, and <math.h>.
